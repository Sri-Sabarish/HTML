# 🌐 Simple HTML Portfolio

This is my **first personal portfolio website**, created while learning **HTML**.

The purpose of this project was to understand and practice the core HTML concepts by building a complete webpage from scratch.

## 📌 What I Practiced

- Basic HTML document structure
- Headings, paragraphs, and text formatting
- Images and hyperlinks
- Tables and lists
- Forms and input elements
- Semantic HTML elements
- Inline styling
- Internal page navigation (Back to Top)
- External links (LinkedIn & Email)

> **Note:** This project was built purely for learning and practice. It is intentionally simple and does not include CSS or JavaScript.

## 🚀 Future Improvements

- Add CSS for a modern and responsive design.
- Improve accessibility and semantic structure.
- Add JavaScript for interactivity.
- Deploy the portfolio online.
